Welcome to your Remotion project!

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm run dev
```

On Windows, Remotion is started through a wrapper that mirrors the project
into `C:\tmp\miniprogram-1-remotion-runtime` before launching the CLI. On
macOS and Linux, the Node wrapper runs Remotion directly, so no PowerShell is
needed.

**Render video**

```console
npm run render -- MyComp out\demo.mp4
```

**Upgrade Remotion**

```console
npx remotion upgrade
```
